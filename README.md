# E-commerce Back End


## Description

A project using Sequelize and object relational mapping to create and maintain an SQL database that represents the backend of an e-commerce site.

## Packages
- dotenv
- Express.js
- mysql2
- Sequelize
- nodemon

## Languages and Tools Used
- Javascript
- ES6 - ECMAScript 6
- Node.js
- MySQL
- Insomniac
